# Portfolio Page

A **personal portfolio website** built with **Vue 3**, **Vite**, and **PrimeVue**, showcasing projects, skills, and technology stack.

**Live Demo:** [https://portfolio-e0zu.onrender.com/](https://portfolio-e0zu.onrender.com/)


## Running the Project Locally

### Frontend
Clone the repository:

```bash
git clone ...
cd portfolio
```

Install dependencies:
```bash
npm install
```

Run locally:
```bash
npm run dev
```
